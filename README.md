## MiniPTJ_Bit
비트캠프학원 교육 과정에서 진행한 개인 프로젝트 입니다.
### 적용기술
Java 기반 MVC2 Pattern을 적용하였고 Spring, MyBatis Framework를 활용하였습니다.
### 기능
엔드유저는 회원가입 후 상품을 구매 할 수 있고 관리자는 회원관리 및 상품 관리를 할 수 있습니다.
